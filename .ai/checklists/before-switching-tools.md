# Before Switching Tools

- [ ] Current task is complete or intentionally paused.
- [ ] Changes are reviewed.
- [ ] `git status` is understood.
- [ ] Next tool has a clear scope.
- [ ] Next tool knows which files it may edit.
- [ ] Next tool knows which files it must not edit.
- [ ] Schema or data work will not be mixed with UI redesign unless explicitly requested.
- [ ] `.local-ai/` remains local-only.

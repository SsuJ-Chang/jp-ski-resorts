# Before Commit

- [ ] Change is focused.
- [ ] No unrelated files are modified.
- [ ] No temporary files are included.
- [ ] `.local-ai/` is not included.
- [ ] Build or typecheck passes when available.
- [ ] No backend, database, auth, paid dependency, runtime API, scraper, or embedded map service was added without a decision record.
- [ ] Data changes include source URLs where needed.
- [ ] Trail map files were not downloaded or redistributed.

# Decision: Initial Region Taxonomy

## Status

Superseded by `.ai/decisions/0006-region-taxonomy-update.md`

## Date

2026-05-19

## Context

The site needs a region taxonomy for discovery, generated counts, region pages, and future static guide visuals.

## Decision

Use this initial ski-relevant taxonomy as a starting point:

- Hokkaido
- Tohoku
- Kanto-Koshinetsu
- Hokuriku
- Chubu
- Kansai / Chugoku

## Reasons

- The grouping is understandable for trip planning.
- It avoids overly granular prefecture-only navigation.
- It can support resort counts and region summaries.

## Consequences

- Some resorts may need careful assignment.
- The taxonomy should be revisited after sample data is created.
- Any accepted change should update project docs and this decision record or a superseding record.
